# Subproblem 4: Submission Checklist

## 1. Goal
Create a final checklist for ECS273 Canvas and GitHub submission so no individual penalty is triggered.

## 2. Why this step exists
The rubric has individual-submission penalties even for team work. The GitHub URL, report PDF, and YouTube URL each need separate attention.

## 3. Files involved
- `target/ecs273/04-Final-Report (1).md` - report submission rules.
- `target/ecs273/05-Final-Presentation (1).md` - video submission rules.
- `target/ecs273/06-Implementation (1).md` - GitHub URL submission rules.
- `README.md` - repository entry point.
- final report PDF - expected filename `teamXXfinal.pdf`.

## 4. Exact changes
Make a checklist with these items:
- GitHub repository URL submitted individually on Canvas.
- README includes description, installation, and execution.
- Final report PDF uploaded individually to Canvas.
- Report PDF filename is `teamXXfinal.pdf`.
- Report includes effort distribution.
- Final video uploaded as an unlisted YouTube video.
- YouTube title is `teamXXpresentation`.
- YouTube URL submitted individually to Canvas.
- URL works in incognito or logged-out browser.
- Team communication and contribution statement are transparent.

## 5. Out of scope
- Do not push to GitHub without user approval.
- Do not upload to Canvas or YouTube from this agent.
- Do not change team member names or effort distribution without user-provided facts.

## 6. Done condition
This step is done when every checklist item is either complete or explicitly marked as needing the user's manual action.

## 7. Verification
Manual checks:
- Open Canvas submission pages and verify uploads/links.
- Open GitHub URL from a logged-out browser.
- Open YouTube URL from an incognito or logged-out browser.

## 8. Expected output
- A final submission checklist, likely in the report working folder or in a short note near the final artifacts.

## 9. Notes for the next step
After this step, ECS273 should be ready for final submission.

## 10. Risks or ambiguity
The agent cannot verify Canvas submission unless the user provides screenshots or exported confirmation. The user must perform final upload checks.
